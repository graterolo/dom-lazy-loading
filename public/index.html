Practica del Curso de Manipulacion del Dom - Platzi

Workshop2 - Usando Lazy Loading

Este proyecto es del curso de manipulacion del dom de Platzi, donde aprendemos a usar web APIS, implementando una API donde obtenemos imagenes random de zorros.

Presiona el boton gris para agregar una imagen de un zorro y el boton rojo para limpiar la pantalla.
